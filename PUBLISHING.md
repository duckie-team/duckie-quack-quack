# Bump

모든 배포 과정에는 PR 의 label 이 중요하게 작용합니다.

### BumpType `#E99695`

| `BumpType_[]`    | Description          |
|------------------|----------------------|
| `Major`          | Major 버전을 bump 합니다.  |
| `Minor`          | Minor 버전을 bump 합니다.  |
| `Patch`          | Patch 버전을 bump 합니다.  |
| `SetSnapshot`    | Snapshots 버전을 설정합니다. |
| `RemoveSnapshot` | Snapshots 버전을 제거합니다. |
