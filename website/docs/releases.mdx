# Releases (Changelog)

Every release includes a bill of materials updates. Check the [GitHub release](https://github.com/duckie-team/quack-quack-android/releases) page for the BOM version.

## 2.0.0-alpha05

*2023-05-25*

*Target: ui*

This release does not include sugar component updates.

### New

- Add `QuackDefaultTextField` component.
- Added design tokens and decorators related to `QuackDefaultTextField`. Please see the `QuackDefaultTextField` documentation for the added items.
- Add `ExperimentalDesignToken` opt-in.
- `QuackIcon` documentation improvements.
- Add `contentDescription` argument to `QuackIcon`.
- Add `Direction` design token.
- Add padding assertion utility.

### Fixed

- Add contract information to `currentFontScale`.
- Correct typo in `QuackTag` document.

## 2.0.0-alpha04

*2023-05-25*

*Target: material*

### New

- Add `Success` color to `QuackColor`.

### Changed

- `LocalQuackTextFieldColors` renamed to `LocalQuackTextFieldTheme`.
- `LocalQuackTextFieldColors#cursor` renamed to `LocalQuackTextFieldTheme#cursorColor`.
